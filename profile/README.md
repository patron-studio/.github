# Patron Studio

Patron Studio is a creative studio building web, ecommerce, and marketing
tooling for client brands.

This GitHub organisation hosts our internal tooling, client platform code,
and the shared design / engineering systems that power them.

## Shared systems

- **patron-sales-dashboard** — multi-tenant sales + creative platform that
  every client brand logs into.
- **patron-ui** — shadcn-style component registry used across client sites
  and internal tools.
- **patron-emailer** — email marketing toolkit and renderer.
- **renovate-config** — shared dependency-update configuration extended by
  every client repo.
